# AI-Powered Exploratory Data Analysis Assistant

An AI-driven system that automatically analyzes datasets, 
detects data quality issues, recommends preprocessing steps, 
and generates analyst-style insights through an interactive 
Streamlit dashboard.

## Key Features

• Automatic dataset analysis 
• Data quality scoring 
• Outlier detection
• Preprocessing recommendations 
• Machine learning problem detection 
• Feature importance analysis 
• Interactive visualizations 
• AI-generated analyst insights 
• Chat interface for dataset questions

## Tech Stack
Python
Pandas
Scikit-learn 
Streamlit 
Transformers 
Matplotlib 
Seaborn

## Run the Application 
streamlit run app.py 

## Example Workflow 
1. Upload dataset
2. Automatic dataset analysis 
3. Data quality scoring 4. Visualizations and insights
